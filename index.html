lightframe

Simple CSS framework to build fast landing pages

Download Source Download Minified Docs

Lightweight is an understatement

Only clocking in at 4kb unminified, 2kb minified bloat is not something you need to worry about!

Easy Learning Curve

If you're familiar with Bootstrap, you can learn this in a matter of minutes

Support Older Browsers

Can't use Flexbox or Grid in production due to legacy browser support? You can use this no problem.
